## WeatherVue

**WeatherVue** is a minimalist web application designed to provide live weather updates. Users can enter a city name and receive current weather details, including temperature, humidity, and wind speed, sourced from the OpenWeatherMap API.

### Features

- **Real-time weather updates**: Fetch and display current weather data.
- **Dynamic weather icons**: Visual representation of weather conditions.
- **Error handling**: User guidance for mistyped city names.
- **Asynchronous updates**: Weather data refreshed without page reloads.

### Skills Utilized

- HTML
- Cascading Style Sheets (CSS)
- JavaScript
- Canva for design elements
